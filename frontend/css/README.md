# CSS

## In CSS, selectors are patterns used to select the element(s) you want to style.

- some examples

```css

#idSelector {
    color: red;
}

.class-selector {
    background: green;
}
/* tag selector */
p {
    font-size: 100px;
}
/* descendant selector */
.parent p {

}
```
